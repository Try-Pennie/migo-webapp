Pennie AI Chat Widget - Integration Example

Installation

Add this code snippet to your website, just before the closing </body> tag:

<script>
  window.PennieChatConfig = {
    sfdcLeadId: 'YOUR_SALESFORCE_LEAD_ID',
    position: 'bottom-right', // Optional: 'bottom-right', 'bottom-left', 'top-right', 'top-left'
    theme: 'light' // Optional: 'light' or 'dark'
  };
</script>
<script src="https://ai-chatbot-sandbox.vercel.app/embed.js"></script>

Configuration Options

  • sfdcLeadId (required): Your Salesforce Lead ID
  • position (optional): Widget position - 'bottom-right' (default), 'bottom-left', 'top-right', 'top-left'
  • theme (optional): Color theme - 'light' (default) or 'dark'
  • baseUrl (optional): Override the base URL (auto-detected by default)

Programmatic API

Control the widget programmatically using JavaScript:

// Open the widget
window.PennieChat.open();

// Close the widget
window.PennieChat.close();

// Toggle the widget
window.PennieChat.toggle();

// Remove the widget
window.PennieChat.destroy();

Click the button in the bottom-right corner to see the widget in action, or use the buttons above to control it programmatically!

Integration Notes

  • The widget is fully responsive and works on mobile devices
  • Press ESC to close the widget when it's open
  • The widget uses postMessage API for secure communication
  • All conversations are persisted across page reloads
  • The widget loads asynchronously and won't block your page
